# Sustenance

Calorie tracker


## Synopsis

```perl6
use Sustenance;
Sustenance.new(:file<sustenance.toml>).gen-macros;
```


## Description

Analyzes caloric intake from Sustenance TOML log.


## Installation

### Dependencies

- Rakudo Perl6
- [Config::TOML](https://github.com/atweiden/config-toml)
